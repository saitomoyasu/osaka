# ほんまかいなセキュリティRPG

会社向けのセキュリティ教育用HTMLアプリです。

「大阪の人はオレオレ詐欺に引っかからないらしい。ほんまかいなと疑うところから始まる」という発想をもとに、社員が実務で遭遇しやすいセキュリティリスクを、ストーリー形式のRPGとして学べるようにしています。

## ファイル

- `osaka-security-rpg.html`
  - アプリ本体です。
  - ブラウザで開くだけで動きます。
  - サーバーやインストールは不要です。

## 学習内容

このゲームでは、以下のテーマを扱います。

- なりすまし電話
- フィッシングメール
- 情報持ち出し
- パスワードとMFA
- 端末とネットワーク
- インシデント報告
- 複合攻撃への総合判断

## ゲームの流れ

1. マップから章を選びます。
2. ツッコミ師匠の会話を読みます。
3. 怪しい状況に対して、取るべき行動を選びます。
4. 回答後に、赤旗ポイントと正しい対応を確認します。
5. 全章クリア後、ラスボス問題に挑戦します。

## 現在の会社ルール反映

以下の前提で問題を作成しています。

- 電話で機密情報の送付依頼が来ることはない。
- そのような電話は対応せず、上司へ報告する。
- 給与や人事関連の不審メールは、HRに確認する。
- カフェなど公共スペースでの業務は禁止。
- 不審なリンク、MFA通知、USB、外部サービス利用は、自己判断せず社内ルールに従う。

## カスタマイズしやすい箇所

`osaka-security-rpg.html` 内の JavaScript データを編集すると、社内ルールに合わせて内容を変えられます。

- `chapters`
  - 各章の問題文、会話、選択肢、解説、学習ポイント
- `finalBoss`
  - 最終問題
- `educationPlan`
  - 学習内容の説明
- `contentPlan`
  - ゲーム化企画の説明

## 社内展開時に差し替えるとよい情報

- 会社名
- 情報セキュリティ窓口
- HR連絡先
- CSIRTまたは情シス連絡先
- 承認済みツール一覧
- リモートワーク・外出先作業の社内規程
- インシデント報告フロー
